# 類別下注頁 — 撰寫指南

## 一、快速開始

### 你要改哪個檔案

每個隊員只改**一個檔案**：

| 類別 | 檔案 |
|---|---|
| 政治 | `frontend/src/pages/public/market-detail-politics/index.jsx` |
| 運動 | `frontend/src/pages/public/market-detail-sports/index.jsx` |
| 時事 | `frontend/src/pages/public/market-detail-current-affairs/index.jsx` |
| 金融 | `frontend/src/pages/public/market-detail-finance/index.jsx` |

### 開發指令

```bash
cd frontend
npm install          # 第一次才需要
npm run dev          # 啟動開發伺服器
npm run build        # 做完後跑一次確認編譯有過
```

### 怎麼看到你的頁面

從首頁選擇你的類別 → 點擊任一 market 卡片 → 就會進到你的頁面（路由 `/markets/{類別}/{id}`）。

## 二、公版給了你什麼

你的頁面已經被 `<DetailPageTemplate>` 包好了。下列東西**你不用自己寫**，公版已經處理好：

| 區塊 | 說明 |
|---|---|
| **Navbar** | 頂部導覽列（透過 `UserLayout` 父層提供） |
| **Footer** | 底部頁尾（透過 `UserLayout` 父層提供） |
| **Hero** | 上半部標題區，固定寬高與字級，自動顯示「市場 #ID」+ 副標 |
| **右欄 TradePanel** | 固定在下注區右側，滑鼠 hover 有金邊升浮 + 追光特效 |
| **2 欄排版** | 左欄（你的內容）＋ 右欄（TradePanel），小螢幕自動堆疊成單欄 |
| **背景與顏色變數** | 深色背景 `#040404`、卡片底色 `#090909`、金色主色 `#d9aa43`、邊框色 `--border` 等全域 CSS 變數都已在 `base.css` ／ `theme.css` 定義好 |

你唯一要做的事：**在 `<DetailPageTemplate>` 的 children 區塊內，設計你的市場資訊卡與圖表內容。**

---

## 四、DetailPageTemplate Props 表

你的頁面會呼叫 `<DetailPageTemplate>`，以下是你可以傳入的 props：

| Prop | 型別 | 必填 | 說明 |
|---|---|---|---|
| `id` | `string \| number` | 是 | 市場 ID，來自 `useParams()` 的 `:id` |
| `subtitle` | `string` | 是 | Hero 區的副標文字，顯示在「市場 #N」下方 |
| `marketId` | `string \| number` | 否 | 傳給 TradePanel 的 marketId，預設等於 `id` |
| `heroExtras` | `ReactNode` | 否 | Hero 區右側的額外內容，例如天氣頁放即時天氣卡片 |
| `children` | `ReactNode` | 否 | **左欄的全部內容**：市場資訊卡、圖表、相關市場連結等 |

### 最簡用法

```jsx
import { useParams } from 'react-router-dom';
import DetailPageTemplate from '../../../components/common/DetailPageTemplate';

export default function PoliticsDetailPage() {
  const { id } = useParams();

  return (
    <DetailPageTemplate
      id={id}
      subtitle="政治事件預測市場"
      marketId={id}
    >
      <div className="trade-market-card">
        {/* 你的市場資訊卡與圖表 */}
      </div>
    </DetailPageTemplate>
  );
}
```

## 六、特效怎麼接

### 6.1 trade-market-card 自動有效果

只要你用 `className="trade-market-card"` 的 `<div>` 包住內容，**自動獲得**：

| 效果 | 觸發 |
|---|---|
| `translateY(-6px)` 升浮 | hover |
| 邊框變金色 `rgba(217,170,67,.35)` | hover |
| 外層光暈 `box-shadow` | hover |
| 鼠標追蹤光暈 `radial-gradient`（`::after` 偽元素） | hover + 鼠標移動 |

這些效果由 `DetailPageTemplate.css` 提供，**不需要寫任何額外程式碼**。

### 6.2 讓自訂區塊也有追光特效

如果你的頁面裡有自訂卡片（例如自己的 CSS class），想加上一樣的鼠標追光：

**步驟 1**：引入 hook

```jsx
import useGlowEffect from '../../../hooks/useGlowEffect';
```

**步驟 2**：在你的元件內呼叫，傳入 CSS selector

```jsx
export default function PoliticsDetailPage() {
  const { id } = useParams();
  useGlowEffect('.my-custom-card');   // ← 新增這行

  return (...);
}
```

**步驟 3**：在你的 CSS 檔加入 `::after` 偽元素規則

```css
.my-custom-card {
  position: relative;
  overflow: hidden;
}

.my-custom-card::after {
  content: "";
  position: absolute;
  inset: 0;
  background: radial-gradient(
    circle at var(--mouse-x, 50%) var(--mouse-y, 50%),
    rgba(217, 170, 67, 0.16),
    transparent 36%
  );
  opacity: 0;
  pointer-events: none;
  transition: opacity 0.3s;
  z-index: 1;
}

.my-custom-card:hover::after {
  opacity: 1;
}
```

這樣你的自訂卡片就會有跟 `trade-market-card` 一樣的鼠標追光效果。

### 6.3 金邊 hover 升浮特效

如果你希望自訂卡片也有金邊 + hover 升浮，在 CSS 加上：

```css
.my-custom-card {
  background: #090909;
  border: 1px solid var(--border);
  border-radius: 28px;
  padding: 32px;
  transition: 0.3s;
}

.my-custom-card:hover {
  transform: translateY(-6px);
  border-color: rgba(217, 170, 67, 0.35);
  box-shadow:
    0 0 0 1px rgba(217, 170, 67, 0.12),
    0 20px 45px rgba(0, 0, 0, 0.45),
    0 0 38px rgba(217, 170, 67, 0.12);
}
```

## 七、CSS 放在哪

### 7.1 三個選擇(正常使用B)

| 做法 | 適用場景 | 範例 |
|---|---|---|
| **A. 不寫 CSS** | 全部用 `base.css`／`DetailPageTemplate.css` 已有的 class（`trade-market-card`、`trade-status-card` 等） | 政治／運動／時事／金融佔位頁目前就是零 CSS |
| **B. 加一個自訂 CSS 檔** | 類別頁有大量自訂樣式（圖表容器、特有卡片） | 天氣頁的 `WeatherDetailPage.css`，約 200 行 |
| **C. 用全域 class** | 小量微調，直接在 JSX 內 `style={{...}}` | 調整 margin、padding |

### 7.2 如果選 B：建立 CSS 檔

1. 在你的類別資料夾下建立 CSS 檔，**檔名跟 JSX 不同就不會衝突**：

   ```
   frontend/src/pages/public/market-detail-politics/
   ├── index.jsx
   └── PoliticsDetailPage.css    ← 新建
   ```

2. 在 `index.jsx` 頂部引入：

   ```jsx
   import './PoliticsDetailPage.css';
   ```

3. CSS 檔內容只放你類別才有的樣式，**不要重複定義** `trade-market-card`、`trade-panel` 等公版已處理的 class。

### 7.3 全域 CSS 變數速查

這些變數已在 `base.css` 定義，你在任何 CSS 檔都可以直接用：

```css
:root {
  --bg: #040404;              /* 頁面背景 */
  --card: #090909;            /* 卡片背景 */
  --gold: #d9aa43;            /* 金色主色 */
  --gold-soft: #b88b2f;       /* 暗金 */
  --green: #00d66f;           /* YES / 漲 / 正數 */
  --red: #ff476d;             /* NO / 跌 / 負數 */
  --text: #ffffff;            /* 主文字色 */
  --text-secondary: #8f8f8f;  /* 次要文字色 */
  --border: rgba(255, 255, 255, 0.06);  /* 卡片預設邊框 */
}
```

### 7.4 RWD Media Query 範例

你的自訂區塊需要用 media query 控制不同螢幕尺寸。寫在你的 CSS 檔最底部即可：

```css
/* 平板以下：圖表單欄 */
@media (max-width: 992px) {
  .politics-chart-grid {
    grid-template-columns: 1fr;
  }
}

/* 手機：內容全寬 */
@media (max-width: 576px) {
  .politics-chart-grid {
    grid-template-columns: 1fr;
    gap: 12px;
  }
}
```

公版提供的 2 欄排版（左內容 + 右 TradePanel）在螢幕 ≤ 1200px 時會自動堆疊成單欄，不需要另外處理。

### 7.5 CSS 命名建議

用**類別前綴**避免跨類別污染：

| 類別 | 前綴 |
|---|---|
| 政治 | `.politics-` |
| 運動 | `.sports-` |
| 時事 | `.current-affairs-` |
| 金融 | `.finance-` |

例如：`.politics-chart-card`、`.sports-scoreboard`。
